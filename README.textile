h1. OneBusAway Java API

This library provides a client for "OneBusAway":http://www.onebusaway.org/ API webapp. You find more informations about OneBusAway on the Google Code "project page":http://code.google.com/p/onebusaway/.

* onebusaway-java-api is a pure Java implementation and relies on @org.json:json-20070829@ and @org.apache.httpcomponents:httpclient-4.0.2@
* onebusaway-android-api uses the same source code but depends on @com.google.android:android-2.3.3@
